注：服务端，运行在win操作系统上的php，程序运行在win服务器上的

一、服务端有如下特点功能
1、即时通信转发信息使用，使用即时通信必须使用。
2、计划任务运行功能，在后台【系统→系统工具→计划任务】设置
3、异步处理任务。
4、定时队列异步处理功能。


二、简单使用(推荐)
1、随便找个目录解压一下，双击start.bat就可以了，不要去管第一行提示，不要做任何修改的，打开了页面就不要关闭了，关闭了就是停止了。
2、目录Rock/push 必须有写入文件权限。
3、IP黑名单，禁止连接，可到Rock/Rockblacklist.php下添加


三、使用设置
1、进入信呼管理后台到【系统→即时通信管理→服务器设置】
   1) 通信地址填写：ws://127.0.0.1:6552/，如果不是本地运行，通信地址请写真实服务器IP
   2) 服务端推送地址填写：http://127.0.0.1:6553/，默认就用这个，不用改。
   3) recID号填写：rockxinhu
2、注127.0.0.1代表你服务端ip地址。


四、自定义配置
1、主要用于内网访问配置，修改默认端口号。
2、修改config.php对应的配置，打开文件按照说明修改。
3、进入信呼后台修改服务器设置


五、运行/停止
1、无界面运行，双击“startserver.vbs”
2、有界面运行，双击“start.bat”
3、停止服务运行，双击“stopserver.bat”
4、加入系统服务，双击“创建加入系统服务.vbs”，然后到addserver文件夹操作。
